# 缘的个人小助手

## AI 助手数据边界

- 两台手机只同步 AI 聊天和临时中转资料。
- 账本、预算和任务历史仍只保存在各自浏览器本地，AI 模块不导入账本代码。
- 原始聊天和临时文件保存在 EdgeOne，处理成功后保留 7 天再删除。
- 完整本地知识库不会上传；网页 AI 只读取 Codex 生成的精简记忆包。
- 联网搜索只发送当前问题的精简查询词，并显示主要来源和日期。

一个免费托管的手机网页，包含只读的今日任务告示栏和设备本地账本。

## 隐私和数据

- 账目保存在当前浏览器的 IndexedDB 中，不会上传到网站或 GitHub。
- 月预算、自定义分类和常用账目也只保存在当前浏览器。
- 最近 7 天任务历史只保存在当前浏览器，不会上传；网页只提供当前有效周总结。
- 清除浏览器网站数据、卸载浏览器或更换手机可能造成账目丢失，请定期使用“导出备份”。
- 网上的 `data/today.json` 只包含生成后的当天任务，不应包含知识库原文、联系方式、账号信息或详细财务数据。
- 本地知识库不会作为网页仓库的一部分发布。

## 本地运行

在项目目录运行 `npx serve . -l 4173`，然后打开 `http://localhost:4173`。

## 更新任务

只替换 `data/today.json`。如果新计划读取、生成或验证失败，保留上一份有效文件。

发布前运行 `npm run validate:today`。只有当天计划和周总结都校验通过时才能提交到网页仓库。版本 2 账本备份包含账目、预算、自定义分类和常用账目；旧版备份仍可导入。
