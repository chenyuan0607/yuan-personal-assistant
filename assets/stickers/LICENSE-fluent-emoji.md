# Microsoft Fluent Emoji 素材说明

本目录内 `assets/stickers/fluent/*.png` 来自 Microsoft Fluent Emoji 官方开源项目。

- 项目地址：https://github.com/microsoft/fluentui-emoji
- 许可证：MIT License
- 用途：作为个人小助手网页的内置表情素材。

后续如果加入你自己上传的表情包，建议单独放入个人素材目录，并记录来源，避免混用不明版权素材。
